# H2O Problem

1. Hydrogen atoms (H) and oxygen atoms (O) arrive at the factory.
2. We only want to bond 2 H and 1 O.
3. Atoms only start bonding when 2 H and 1 O arrives.
4. Only 1 bonding can occur at a time.
